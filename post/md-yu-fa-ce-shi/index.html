1. 图片作为视频预览的md语法套用

我们可以把这个 Markdown 语法拆解成最基础的两个积木，你一看就懂了：

🧱 积木 A：普通的链接
Markdown 里的链接公式是：“方括号包文字，圆括号包网址”。

语法： 显示的文字
例子： 点击这里去百度

逻辑： 这里的 [ ] 是皮（给人看的），( ) 是馅（给浏览器用的）。

🧱 积木 B：普通的图片
图片的语法跟链接几乎一模一样，唯一的区别是前面多了一个感叹号 !。

例子：

逻辑： 感叹号 ! 的意思是 "嘿！这里要展示出来！" (Show Image)。如果没有感叹号，它就只是一个普通的文字链接。

🍔 合体：把图片“塞进”链接里
现在的任务是：我们要让**“皮”**（给人看的东西）不再是枯燥的文字，而是一张图片。

操作步骤： 拿过 [积木 A]，把它的 [显示的文字] 挖掉，把 [积木 B] 整个塞进去。
